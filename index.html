Array.prototype.some()

Метод массива

Цель

Этот метод массива помогает вам определить, соответствует ли одно или несколько его значений вашему условию

Если вам это кажется не очень понятным, позвольте мне проиллюстрировать это на примере...

Вот список рандомных сотрудников:

            const operatives = [
                { id: 12, name: 'Baze Malbus', guy: false },
                { id: 44, name: 'Bodhi Rook', guy: true },
                { id: 59, name: 'Chirrut Îmwe', guy: false },
                { id: 122, name: 'Jyn Erso', guy: false }
            ];

Вы хотите знать, есть ли среди ваших сотрудников хотябы один 'пилот'

Есть много способов достичь этой цели. Большинство людей, вероятно, будут использовать .forEach() и делать что-то вроде этого:

            let listHasPilots = false
            operatives.forEach(function (operative) {
                if (operative.guy) {
                     listHasPilots = true
                }
            });

Это много строк, чтобы увидеть, содержит ли список 'пилотов', и guy !== gay

Давайте попробуем использовать .some()

            const listHasPilots = operatives.some(function (operative) {
                 return operative.pilot
            });

А еще короче?

            const listHasPilots = operatives.some(
                 operative => operative.pilot )

Как это работает?

  1. Вы передаете в .some() функцию в качестве аргумента.
  2. Эта функция выполняется для каждого значения в массиве.
  3. Затем вы можете увидеть, соответствует ли значение написанному вами условию (true или false).
  4. Как только функция вернет первый true, .some() сама вернет true
  5. Если ни одно из значений при обработке в ваших условиях не возвращает true, то .some() возвращает false.

Обратите внимание, что как только возвращается первое значение true, .some() прекращает проверку других значений массива.

В нашем примере выше, функция запускается только дважды, поскольку второй сотрудник, Бодхи Рук, является пилотом. Не нужно проверять других пилотов.

            const operatives = [
                { id: 12, name: 'Baze Malbus', guy: false },
                { id: 44, name: 'Bodhi Rook', guy: true },
                { id: 59, name: 'Chirrut Îmwe', guy: false },
                { id: 122, name: 'Jyn Erso', guy: false }
            ];

И так

Если некоторые из ваших значений массива соответствуют тому, что вы ищете, .some() вернет true. Если нет, он вернет false.
